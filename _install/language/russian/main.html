Прежде чем начать процесс установки Вы должны получить информацию о сервере MySQL: имя сервера, Ваш логин и пароль для соединения с сервером.

Если у вас нет подобной информации, обратитесь к системному администратору или в организацию, которая предоставляет Вам услуги хостинга.

Минимальные требования к програмному обеспечению устновленному на сервере:

  • PHP 5.16
  • MySQL 4.1.20
  • файл mainfile.php в корневой директории должен быть доступен для записи.

В дальнейшем программа установки проверит эти данные.
