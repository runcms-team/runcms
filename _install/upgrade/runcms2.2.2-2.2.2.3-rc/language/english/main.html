Welcome to RUNCMS update to version 2.2.3.0 RC!

To update RUNCMS - you must replace the previous version of the files are on a distribution upgrade, as well as make changes to the database table.

Before moving to the new version, we strongly recommend that you make a backup as all files and folders site and database tables (см.:  phpmyadmin)!
