Администрирование заголовков

Этот модуль позволяет Вам контролировать поступающую информацию на Ваш сайт. С этим модулем Вы сможете встроить внешнюю информацию с других сайтов прамо в Ваш сайт.

Добавить/Редактировать заголовок:

Имя сайта
Имя сайта которое Вы хотите дать заголовку: Только для личного использования.

URL сайта
URL сайта с которого Вы получаете заголовок. Для личного использования.

URL к файлу RDF/XML
Здесь необходимо указать полный путь к файлу заголовков. Весьма часто этотот файл называется backend.php в корневой директории удаленного сайт.

Шаблон
Шаблоны в которых будут показаны заголовки на Вашем сайте. Некоторые шаблоны показывают только одну строку, например только заголовок (simple.txt), другие показывают заголовок и кораткое изложение (description.txt).

Примечание: Вы можете редактировать шаблоны под свои требования в /modules/headlines/templates/
Это также зависит от того доступно-ли описание с удаленного сайта или нет.

Тип
Эта опция позволяет Вам контролировать будут-ли заголовки показаны только в блоке (см.: admin/blocks) или они также будут показаны на главной странице с заголовками. По этому Вы можете показывать разные заголовки на каждой из них.

Минут кэш .
Время кэшированния заголовков до обновления данных (в минутах). Для увеличения скорости работы.

Maкс. кол-во
Максимальное количество заголовков для показа.

Статус:
Активные заголовки будут показаны на Вашем сайте, не активные соответственно нет.

Для активных заголовков Вы можете редактировать его порядок и затем подтвердить кликнув на Переупорядочить.
Это указывает на порядок отображения заголовков в блоке/странице.
