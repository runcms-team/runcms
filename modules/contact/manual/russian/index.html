Администрирование модуля
"Информация / Контакты"

Редактировать раздел "О нас"

Отправлять подтверждение?:
Если выбрано, то автоматически отправителю будет выслано подтверждающее письмо.

Разрешить Attachments?:
Если выбрано, то отправитель сможет присоединять к письму разрешенные типы файлов.

Разрешить Анонимам Attachments?:
Если разрешен attachments, будет-ли разрешено Анонимам присоединять файлы?

Max. размер Attachment:
Максимальный размер присоединяемого файла (в байтах).

Разрешенные типы файлов :
Разрешенные типы файлов для присоединения к письму, разделенные символом "|" (без кавычек)

BCC To (копия):
Разделённый запятой список email адресов которым также будет отправлена копия сообщения.

Причина контакта:
Список причин для контакта разделенный символом "|" (без кавычек). Если болше 1 причины контакта в списке, пользователю представится выпадающий список, в другом случае будет текстовое поле в котором можно будет указать другую причину.

Вступительный текст :
Вступительный текст (если заполнено) который будет показан наверху страницы "О нас".

Редактировать раздел "Связь с нами"

Вступительный текст :
Вступительный текст (если заполнено) который будет показан наверху страницы "Связь с нами".

Редактировать раздел "Вебмастеру":

Текст в Alt:
Текс показываемый когда пользователь наведёт курсор на изображение.

Заголовок:
Применяется только для текстовых ссылок. Если это поле заполнено.

Ссылка на изображение:
Любое имя локального файла находящегося в каталоге contact/cache/images/ или полный тупть URL к существующему изображению.

Свой код:
Необязательно. Html, javascript, flash code, и т.д. Если поле заполнено, все опции выше отменяются.

Примечание: Любые незаполенные поля будут скрыты от пользователей. Для примера: если у Вас незаполена информация "О нас", пользователь увидит простую станицу для контактов, если у Вас нет баннеров и ссылок,то этот блок не будет показан, и так далее...

Вы также можете удалять/добавлять подменю в Вашем модуле контактов изменяя файл xoops_version.php в корневой директории модуля и изменяя/удаляя /комментируя строки:
$modversion['sub'][xx]['name'] = ....
$modversion['sub'][xx]['url'] = ....

Если Вам надо настроить высылаемые Вам/пользователям на email сообщения, изменяйте файлы в: language/russian/mail_template/*
Добавя тэг {IS_HTML} в файл шаблона для сообщения, Вы тем самым измените тип отправляемого e-mail на HTML.
