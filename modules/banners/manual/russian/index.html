Администрирование баннерной системы

Контроль текущей ротации баннеров на Вашем сайте. Вы можете активировать баннерную систему в: администрироание/общие настройки

Текущие активные баннеры
Все баннеры показываемые в текущий момент на Вашем сайте, со статистикой для каждого из них.

Завершенные
Все баннеры у которых истекло время показов или количество показов, здесь Вы можете добавить количество показов или удалить их совсем.

Рекламодатели
Ваши клиенты имеющие аккаунт и подписанные на ротацию баннеров. Здесь Вы можете редактировать информацию клиентов или удалить их.

Добавить баннер
Позволяет добавлять баннер для ротации для определенного клиента.

Имя клиента:
Просто выберите имя клиента подписанного на ротацию баннеров.

Куплено показов:
Добавить количество показов или 0 для бесконечного показа .

Показывать как:
Когда баннер вставляется в блок и т.д., просто вызывается showbanner("CC"); например: если Вам необходимимо показывать баннеры только в выбранном блоке (Center Center). Значение по умолчнию "N".

N = Обычный баннер (* по умолчанию)
A = Показывать везде
BA = Блоки: ВСЕ

Панель блоков:
SL = Левая
SR = Правая
SLR = Левая/Правая

Центральные блоки:
CL = Центр Слева
CR = Центр справа
CLR = Лево/Право
CC = Центр центр
CA = Все

Или Вы можете использовать свою кодировку типа AZERTY тогда будут показаны баннеры только с этой кодировкой.

Примечание: это вспомогательная настройка, если Вы хотите, чтобы таким образом определенные баннеры появились в блоке, убедитесь, что Вы вызываете showbanner() правильно.

URL картинки баннера:
Введите имя картинки находящееся на Вашем сайте, полный путь URL к картинке на удаленном сайте или выберите картинку для загрузки на Ваш сайт.

ALT картинки:
Любой ALT текст (когда посетитель наведет курсор на баннер) который Вы хотите показать вместе с баннером.

URL ссылки:
Куда будет направлен посетитель после клика на баннер.

Свой код : *
HTML, JavaScript, и т.п. ..используется для вставки готового кода для flash или баннерных сетей.

Добавить клиента
Позволяет Вам добавить в список нового клиента, после этого Вы сможете добавить баннеры на его аккаунт. Вам следует заполнить всю необходимую информацию включая логин/пароль.

Примечание: Клиенты смогут получить свою статистику в www.yoursite.com/modules/banners/ введя логин/пароль указанный Вами.

Помните, что даже если Вы и дадите доступ к модулю для анонимных пользователей (и всех остальных) и обращении их к модулю, модуль вызовет ошибку access denied! (доступ закрыт!)
