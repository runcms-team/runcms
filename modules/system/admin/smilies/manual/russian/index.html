Управление смайлами

В данном разделе администрирования сайта можно управлять смайлами (графические иконки, выражающие эмоции) - создавать/редактировать/удалять.

Код:
Текстовый код смайла. Данный код в тексте будет заменен графическим изображением.

Смайл:
Изображение, заменяющее код.

Описание:
Краткое описание эмоции характеризующей смайл.

Действие:
Редактировать или удалить смайл.

Добавить смайл:

Код смайла:
Текстовый код смайла.

URL смайла:
Любое имя и локальная картинка в папке /images/smilies/, полный URL на внешнюю картинку или же выберите картинку для закачки.

Описание смайла:
Краткое описание смайла.
