Поиск пользователей

В этом разделе, Вы можете по определенным параметрам, искать в базе данных пользователей.  Редактировать или удалять их.

Используйте эту форму для удаления неактивных пользователей, активации ожидающих подтверждения регистрации или простого поиска необходимых Вам пользователей и информации о них.

Осуществлять поиск можно по всем элементам пользовательской учетной записи (Имя пользователя, E-mail, Интересы и т.д.) Более подробно об этих элементах смотрите в руководстве к разделу: Администрирование/Редактировать пользователей

Каждому из этих критериев можно задать определенные условия точности нахождения:

Точное совпадение:
Допускается только точные совпадения между элементом учетной записи пользователя и введенным словом/числом.

Начинается с:
Элемент учетной записи пользователя должен начинаться с указанного.

Оканчивается на:
Элемент учетной записи пользователя должен оканчиваться на указанное.

Включает:
Означает, что элемент учетной записи пользователя (Имя пользователя, E-mail, Интересы и т.д.), может содержать, введенное Вами слово или число (если это например номер ICQ).

Кроме того можно установить фильтр по следующим критериям:

Последний вход пользователя - более чем X дня(ей) назад
Последний вход пользователя - менее чем X дня(ей) назад
Дата регистрации пользователя - более чем X дня(ей) назад
Дата регистрации пользователя - менее чем X дня(ей) назад
Число сообщений пользователя - более X
Число сообщений пользователя - менее X
Статус пользователей (только активные, только неактивные, все пользователи)

Найденные результаты можно представить в следующем виде:

Сортировать по (имени пользователя, E-mail, последнему входу, дате регистрации, числу сообщений)
Порядок (по возрастанию, по убыванию)
Число пользователей на страницу

После нахождения результата, будет выведен список найденных пользователей. Помимо просмотра персональных данных об этих пользователях, выбрав из них нужных Вам, можно удалить этих пользователей, а так же послать им письмо или приватное сообщение.
