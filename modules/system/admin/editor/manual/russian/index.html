Конфигурация редактора FCKeditor


  • Активировать редактор
    • Включение редактора. Если эта опция выключена, то редактор не будет доступен ни для кого.


  • Активировать редактор для пользователей
    • Редактор будет активирован для всех пользователей.


  • Активировать редактор для раздела "E-mail пользователям"


  • Активировать интеграцию редактора на JavaScript?
    • Рекомендуется использовать эту опцию, ускаряет загрузку редактора
    если эта опция выключена то редактор будет интегрирован на PHP.


    Настраевая следующие параметры незабудьте что, каждый пользователь входит минимум в 1 группу "Пользователи" тоесть в группу "Пользователи" входят все зарегистрированные пользователи, включая Админов и всех созданных вами групп.
    Если вы хотите выбрать несколько групп удерживайте клавишу "CTRL"

  • Укажите группы, имеющие право пользования встроенным в редактор Файловым-менеджером.
    • в редактор встроен файловый менеджер, нужно выбрать какя группа пользователей имеет к нему доступ
    следующие 3 пункта параметра завист от этого пункта. в них можно более точно определить права той или иной группе.


  • Укажите группы, имеющие право загружать файлы на сервер.
    • С помощью FCK можно осуществлять прямую закачку файлов на сервер. Эта опция определяет кому можно это делать. файлы будут сохранятся в специально отведённой для этого директории "http://ваш_сайт/UserFiles/..."
    если не пользоваться файл менеджером то они сами будут сортироваться.
    для Link -> папка "Files"
    для Images -> "Images"
    для Flash -> "Flash"
Также в начиная с версии cms 1.5 в редектор была добавлена кнопка [pagebreak] разделение контента на страницы.
Также возможно создание кнопок с любым другим функционалом.

В файл менеджер возможно установить функции переименования, удаления , назначения прав доступа,
к файлам и папкам рекурсивно , предворительный просмотр например картинок.
