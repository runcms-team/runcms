Управление блоками

В этом разделе, Вы можете осуществлять полную настройку блоков вашего портала. Подключать, выключать блоки, добавлять новые различные типы блоков, редактировать настройки блоков, копировать, определять местонахождение блока.

Блоки портала делятся на системные и блоки конкретных модулей. Системных блоков всего 6: динамическое меню, блок поиска, блок авторизации, обновления на сайте, информация о сайте и блок пользователя. Блоки модулей появляются в этом разделе только после установке этого модуля в системе. После удаления модуля, удаляются и все его блоки (если они есть)

Подключение блока

Чтобы подключить нужный блок, Вы должны выбрать значение поля "Видимость"  конкретного блока равным "Да" после этого нажать на кнопку "Подтвердить". Чтобы отключить соответствующий блок, сделайте то же выбрав значение "Нет".

Заметка: После подключения блока Вы должны настроить права доступа к нему в разделе "Администрирование/Управление группами" или же перейдя в раздел редактирования этого блока. Системные блоки с внутренними функциями (например блок "Логин") будут доступны сразу.

Типы блоков

На данный момент существуют 8 типов блоков: боковые блоки (правые и левые) и центральные блоки (верхние и нижние), верхние и нижние центральные блоки, в свою очередь делятся на правые, центральные и левые.

Позиция блоков

Положение каждого однотипного блока по отношению к другому контролируется параметром "Позиция" Числовое значение этого параметра показывает следование блоков друг за другом.

Заметка: Центральные блоки могут быть размещены в любом порядке. Но нужно иметь в виду, что левый блок всегда появляется вперед правого. Если у вас есть 2 блока на одной строке, убедитесь, что значение параметра "позиция"  левого блока меньше "позиции" правого и сделайте так, чтобы ни один блок из этой области больше не имел такого значения.

0. Центральный блок
1. Левый блок
2. Правый блок
3. Центральный блок
4. .......

Расположение блоков в рамках конкретных модулей

Каждый из блоков может быть ограничен расположением в рамках какого либо конкретного модуля. Для этого нужно в раскрывающемся списке поля "Расположение" выбрать необходимый модуль (по умолчанию блоки показываются в рамках всех модулей.

Копирование блоков

Каждый из блоков может быть скопирован, для этого нужно нажать на кнопку "Копировать" соответствующего блока. После этого мы получим совершенно независимый блок обладающий всеми свойствами исходного.

Добавление новых блоков

При желании мы можем создать новый блок содержащий необходимую нам информацию.

Сторона блока: Укажите тип блока: боковой блок (правый и левый), центральный... и т.д.
Расположение: Укажите модуль в рамках которого этот блок будет показан (или выберете опцию "все" модули).
Позиция: Положение блока по отношению к другим (более подробно смотрите выше)
Видимость: Подключить этот модуль или нет.
Заголовок: Заголовок над блоком.
Контент: В этом поле мы размещаем нужную нам информацию. Это может быть текст, текст в формате HTML, JavaScript или код PHP. В зависимости от содержания этого поля, вы должны настроить нижеследующую опцию.
Тип контента:
  HTML - текст в формате html, может содержать JavaScript и т.д..
  PHP - php-код, важно, чтобы он не содержал <?php и ?> теги внутри блока
  Автоформат (смайлы отключены) - автоматически преобразует символ "новая строка" в тег <br>, смайлы показаны не будут. С учетом вышесказанного можно использовать как просто текст так и HTML
  Автоформат (смайлы включены) - то же что и выше, но с показом смайлов.

В завершении нажимаем кнопку "Подтвердить"

Редактирование блока

Нажав кнопку "Редактировать" конкретного блока Вы сможете отредактировать дополнительную информацию о блоке. Помимо опций описанных выше (сторона блока, расположение, позиция, видимость, заголовок, контент, тип контента) Вы можете настроить следующие опции:

Показать на странице: здесь можно выбрать показывать блок только на главной странице сайта (значение "Стартовой:"), только на главной странице выбранного модуля (значение "1-й странице модуля:") или на всех страницах (значение "Других страницах:"). Плюс опция - показывать блок на «специальной» странице (значение "Специальной:"). Под специальными страницами подразумеваются страницы, не относящиеся к модулям - страница поиска, личных настроек пользователя, регистрации, авторизации и т.д.
Установить доступ: В этой опции мы устанавливаем доступ для различных групп для этого модуля. Соответственно, пользователи которые входят в группу которая не будет выбрана в этом списке, данный блок не увидит. Доступ к блокам так же можно редактировать  в разделе: Администрирование/Управление группами
Позиция дополнительного контента: Данные которые мы введем в поле контент можно располагать как выше основной информации блока, так и ниже.

Отредактированные настройки блока необходимо сохранить. Так же мы можем выбрать опцию "Предварительный просмотр" - будет показан редактируемый блок с учетом введенных изменений.
