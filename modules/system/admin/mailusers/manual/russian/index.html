Отправка сообщений пользователям

В данном разделе администрирования сайта можно отправить сообщения пользователям или любому другому адресату, как на адрес электронной почты так и приватные.

Кому отправить:
Это фильтр, который позволяет определить группу или пользователя для отправки письма/приватного сообщения.

Критерии выбора (все необязательные):

Группа
Последний вход после определенного времени
Последний вход до определенного времени
Последний вход более чем X дней назад
Последний вход менее чем X дней назад
Только пользователям с проверенной почтой
Послать сообщение только неактивным пользователям
Регистрация после определенного времени
Регистрация до определенного времени

Дополнительный список E-mail, для отправки сообщений:
Чтобы отправить сообщение на дополнительный адрес или несколько адресов электронной почты, введите его (их) в это поле. Если адресов несколько, они должны идти через запятую.

Приоритет E-mail (только email):
Устанавливает степень важности письма (High, Normal, Low и т.д.)

Формат E-mail (только для e-mail):
По умолчанию - текст. HTML позволит отправить письмо в формате html (соответственно можно использовать html теги в самом письме).

Требуется подтверждение (только e-mail):
При получении письма, получателю будет предложено отправить уведомление о его прочтении.

От кого (только e-mail):
Показывает имя отправителя в поле От.

С адреса (только e-mail):
Показывает адрес отправителя.

Тема:
Тема письма/приватного сообщения

Содержание письма:
Текст письма/приватного сообщения.

Заметка: Для украшения письма Вы можете использовать разрешенные HTML теги (если формат письма - HTML).

Прикрепить файл (только e-mail):
Позволяет прикрепить файл к письму.

Дополнение:

В теме письма и его содержании Вы можете использовать специальные теги:

{X_UID} будет заменен на id пользователя
{X_UNAME} будет заменен на имя пользователя
{X_UEMAIL} будет заменен на e-mail пользователя
{X_UACTLINK} будет заменен на ссылку активации для данного пользователя
